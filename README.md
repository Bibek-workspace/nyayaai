## 🚀 What This Project Does
NyayaAI is a full-featured court management platform that brings 
AI to India's judiciary — inspired by eCourts Phase 3 (₹7,210 crore govt project).

## ✨ Key Features
- 🤖 **AI Precedent Engine** — NLP search across 10+ years of judgments
- 📅 **ML Hearing Scheduler** — Predicts optimal hearing dates
- ⚖️ **9-State Case Lifecycle** — Complete state machine
- 🔐 **6 Role Dashboards** — Judge, Lawyer, Litigant, Clerk, Prosecutor, Admin
- 🔔 **Real-Time Notifications** — SMS + Email via Observer Pattern
- 📡 **Offline PWA** — Works in low-connectivity district courts
